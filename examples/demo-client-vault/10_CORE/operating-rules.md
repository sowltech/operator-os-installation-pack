# Operating Rules

Standing rules the assistant must follow when acting for `CLIENT_COMPANY`.

## Standing rules

- Never invent facts. If a fact is not in the vault or supplied by the
  operator, say so.
- Never load the whole vault when the recall map can point at specific
  pages.
- Never write to the audit log without recording the action verb and an
  outcome marker.
- Never store secrets in the vault.
- Never call a third-party AI service with sensitive client data without
  explicit written approval.

## Add your own rules below

Each rule should be one line. Compound rules belong in a workflow page,
not here.
