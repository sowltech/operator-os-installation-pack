# Client Onboarding Workflow

| Step | Trigger | Inputs | Outputs |
|---|---|---|---|
| 1 | New client signs the engagement letter | signed letter | onboarding ticket |
| 2 | Schedule discovery call | calendar | scheduled call |
| 3 | Run discovery call | call template | filled discovery doc |
| 4 | Decide scope and timeline | discovery doc | decision file |
| 5 | Send statement of work | scope decision | sent SoW |
| 6 | Begin delivery | signed SoW | first delivery row in audit log |

Adapt steps to the actual client process before first use.
