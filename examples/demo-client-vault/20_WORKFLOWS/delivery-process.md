# Delivery Process Workflow

| Step | Trigger | Inputs | Outputs |
|---|---|---|---|
| 1 | Signed scope | scope doc | delivery kickoff row |
| 2 | First milestone work begins | scope doc, source notes | first deliverable draft |
| 3 | Review with client | draft | feedback rows |
| 4 | Iterate to acceptance | feedback | accepted deliverable |
| 5 | Final handover | accepted deliverable | handover row in audit log |

Adapt to the actual delivery cadence before first use.
