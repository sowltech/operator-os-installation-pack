# Master Index

This file is loaded **first** by any assistant working against this vault.

## Purpose

Tell the assistant what exists and which 1–5 pages to load for the task at
hand. Loading the whole vault for any task is forbidden by the recall
discipline in `recall-map.md`.

## Retrieval rule

1. Load this index first.
2. Identify the task type.
3. Use `recall-map.md` to choose 1–5 specific pages.
4. Load only those pages.
5. Execute the task.
6. Append a row to the audit log.
