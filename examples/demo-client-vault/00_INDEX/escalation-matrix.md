# Escalation Matrix

Use the **minimum** cognitive level that still satisfies the task.

| Level | Question | Use for |
|---|---|---|
| 1 | What should be said? | writing, content, formatting |
| 2 | What information is required? | research, retrieval, intake |
| 3 | What must persist? | notes, decisions, audit rows |
| 4 | What capabilities are required? | tool selection, skill choice |
| 5 | What sequence creates the outcome? | SOPs, workflows |
| 6 | How should knowledge be organised? | taxonomies, recall maps |
| 7 | What reality governs this task? | businesses, ecosystems |
| 8 | How should intelligence itself be structured? | planning, governance |
| 9 | How does this improve itself? | retrospectives |
| 10 | How do multiple intelligences cooperate? | human + AI collaboration |

See `templates/escalation-matrix-template.md` for the full trigger list.
